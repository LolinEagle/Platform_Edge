# Platform Edge
Platform Edge is a 2D action-adventure platformer that combines shooting mechanics with classic platforming. Set in a pixel art world, players navigate challenging levels filled with obstacles and enemies, using their skills to progress. The game offers a casual, single-player experience with vibrant sprites and smooth gameplay.
